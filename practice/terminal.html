Welcome to How to Make a Website

Steps Before Vim

Hi if this is your first time making a website great. I am going to go through the procces of setting up and making you first website.

  • First you need to make sure you have a github repository, and make sure it's linked to terminal. If you have this done your almost ready.
  • Finally make a file for all your websites to go in like Practice the practice folder is where all your websites will go.

First Website

In the terminal, make sure you're in your practice folder, and then type this into the line

echo "your text" < practice.html Enter

Then type ls and make sure that you have a new file called practice.html

This will add the text to your website you can now visit your website name of repo.github.io/practice/practice.html

Starting vim

Now we are going to be editing our website vim practice.html this is going to allow you to edit your website.

You can type whatever you want on vim though certain things might not work so keep it simple just words.

When your done writing what you want hit escape and type :wq. This will save your work then send you back to terminal.

  • Once you're back in terminal type git add practice.html this will add you updates to github
  • Next you will type git commit -m "add text" this will tell github that you are certain that you want to add your update. REMEMBER TO ALWAYS ADD -M OTHERWISE TERMINAL WILL BE MAD.
  • Finally type git push that pushes your update to github to update your website. ALWAYS REMEMBER TO DO THIS.

Starting HTML

In this you will be typing the most minimal HTML you can type. Later we will add font color and more.

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Homepage</title>
</head>
<body>
    <p>First website</p>
</body>
</html>

This code is the minimal HTML you can type. The building blocks for all websites. Without this you can't link stuff put images, and much more.

What about new lines look below 

<!DOCTYPE html>
<html lang="en">
<head>
<title>Homepage</title>
</head>
<body>
<p>To add new lines</p>
<p>Just add new lesser than and greater than signs with p in them</p>
</body>
</html>
